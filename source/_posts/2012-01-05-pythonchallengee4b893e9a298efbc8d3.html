看来是需要从这段火星文中找出所有的字母就OK了。

将这段COPY出来，保存为一个文件"3-source.txt"，用python做如下运算
[cc lang="python"]
import string

file = open("3-source.txt", "r")
ret = [char for char in file.read() if char in string.letters]
print "".join(ret)
file.close()
[/cc]
